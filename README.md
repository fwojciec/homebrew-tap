# Homebrew Tap

Homebrew formulas for CLI tools by [@fwojciec](https://github.com/fwojciec).

## Installation

```bash
brew tap fwojciec/tap
brew install <formula>
```

Or directly:

```bash
brew install fwojciec/tap/<formula>
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `j4c` | Non-interactive Jira CLI for AI agents |

## Updating

```bash
brew update
brew upgrade <formula>
```
